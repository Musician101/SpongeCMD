# SpongeCMD
Simple command API for SpongeAPI plugins.

Groovy:

```Groovy
repositories {
    maven {
        url = 'https://jitpack.io'
    }
}

dependencies {
    implementation 'com.github.Musician101:SpongeCMD:<version>'
}
```

Kotlin DSL:

```Kotlin
repositories {
    maven("https://jitpack.io")
}

dependencies {
    compileOnlyAPI("com.github.Musician101:SpongeCMD:<version>")
}
```
